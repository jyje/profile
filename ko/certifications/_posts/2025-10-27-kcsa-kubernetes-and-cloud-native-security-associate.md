# KCSA: Kubernetes and Cloud Native Security Associate (2025)

* UNORDERED TOC
{:toc .large-only}

## 1. 자격증 소개

Kubernetes and Cloud Native Security Associate (KCSA) 자격증은 Kubernetes 및 클라우드 네이티브 생태계의 보안 실무와 원칙에 대한 기초 지식을 검증합니다. Cloud Native Computing Foundation (CNCF)과 The Linux Foundation이 공동으로 개발한 KCSA는 클라우드 네이티브 보안 개념과 모범 사례에 대한 이해를 증명하고자 하는 분들을 위해 설계되었습니다.

KCSA 시험은 온라인 감독 시험으로, 90분 동안 60개의 4지선다형 객관식 문제를 풀어야 합니다. 모든 문제는 이론적이며 실습 과제는 없습니다. 합격 점수는 75%이며, 자격증은 2년간 유효합니다. 시험은 다음과 같은 클라우드 네이티브 보안 영역을 다룹니다:

- **Kubernetes 보안 기초**: 핵심 보안 개념, Pod 보안, 네트워크 정책, RBAC
- **컨테이너 보안**: 컨테이너 이미지 보안, 스캐닝, 런타임 보안
- **클라우드 네이티브 보안 도구**: CNCF 생태계의 보안 도구 및 실무
- **보안 모범 사례**: 보안 구성, 시크릿 관리, 컴플라이언스
- **위협 탐지 및 대응**: 클라우드 네이티브 환경의 모니터링, 로깅, 인시던트 대응

CKS (Certified Kubernetes Security Specialist)와 같은 실습 기반 보안 자격증과 달리, KCSA는 이론적 지식과 보안 개념의 이해에 중점을 두어 클라우드 네이티브 보안을 처음 접하는 분들에게 훌륭한 시작점이 됩니다.

## 2. 시험 후기

![KCSA 자격증](/assets/img/certtifications/8f52ead9-2648-4bac-a881-ef8e8bf866dd.png){:.lead width="500" height="100" loading="lazy"}

KCSA 자격증 2025. 2025년 10월 27일 발급. 2027년 10월 27일 만료. 자격증 확인은 [여기](https://www.credly.com/badges/1206ad1c-e348-4328-934d-72e44ca434be)에서 할 수 있습니다.
{:.figcaption}

저는 2025년 10월 27일에 KCSA 시험에 응시하여 합격했습니다. CKA와 CKAD 자격증을 통한 Kubernetes 보안 실무 경험을 바탕으로, KCSA 시험은 클라우드 네이티브 보안 실무에 대한 이론적 이해를 검증하고 체계화하는 훌륭한 방법이었습니다.

시험 형식은 실습 없이 4지선다형 객관식 문제로만 구성됩니다:
- 60개의 문제가 모두 이론적 객관식 문제 (커맨드라인 작업 없음)
- 실제 구현보다는 보안 개념, 원칙, 모범 사례에 초점
- Kubernetes뿐만 아니라 전체 클라우드 네이티브 생태계의 보안 측면을 다룸

문제는 Kubernetes 보안 정책, 컨테이너 보안, 시크릿 관리, 네트워크 보안, 컴플라이언스, CNCF 생태계의 보안 도구 등 다양한 보안 영역을 테스트했습니다. Kubernetes 실무 경험이 도움이 되었지만, 시험은 보안 원칙과 클라우드 네이티브 환경에 적용되는 방식에 대한 포괄적인 이해를 요구했습니다.

시험 준비를 위해 KodeKloud의 [Kubernetes and Cloud Native Security Associate (KCSA)](https://learn.kodekloud.com/user/courses/kubernetes-and-cloud-native-security-associate-kcsa) 강좌를 수강했습니다. 강사는 Mumshad Mannambeth와 Nimesha Jinarajadasa입니다. 이 강좌를 수강하기 위해 KodeKloud Basic 플랜을 구매했습니다 (Pro 플랜은 필요하지 않음). 총 6.78시간의 이 강좌는 클라우드 네이티브 보안 개요, Kubernetes 클러스터 구성 요소 보안, 보안 기초, 위협 모델, 플랫폼 보안, 컴플라이언스 프레임워크를 포함한 모든 시험 영역을 다룹니다. 강좌의 실습 랩, 인터랙티브 콘텐츠, 최신 보안 정보에 대한 정기적인 업데이트가 특히 유용했습니다. KodeKloud는 실습을 위한 실습 랩 환경을 제공하지만, 실제 KCSA 시험은 실습 없이 4지선다형 객관식 문제로만 구성됩니다. 강좌의 인터랙티브 퀴즈와 모의 시험은 보안 개념을 강화하고 실제 시험 형식에 대비하는 데 도움이 되었습니다. 체계적인 학습 경로와 지속적인 콘텐츠 업데이트 덕분에 시험을 순조롭게 통과할 수 있었습니다.

## 3. 시험 준비 팁

제 경험을 바탕으로, KCSA 시험을 준비하는 분들을 위한 중요한 팁을 공유합니다:

**시험 전 필수 리소스:**
- **공식 CNCF 커리큘럼**: 공식 KCSA 커리큘럼을 검토하여 정확한 시험 영역과 비중을 파악하세요
- **CNCF 보안 랜드스케이프**: [CNCF Landscape](https://landscape.cncf.io/)의 보안 관련 프로젝트에 익숙해지세요 - 보안 도구, 목적, 클라우드 네이티브 보안 생태계에서의 역할을 이해하세요. 랜드스케이프는 성숙도 수준(Graduated, Incubating, Sandbox)별로 분류된 보안 프로젝트의 포괄적인 뷰를 제공합니다
- **CNCF 블로그**: [CNCF 블로그](https://www.cncf.io/blog/)를 정기적으로 확인하여 최신 보안 업데이트, 모범 사례, 새로운 보안 프로젝트 발표를 파악하세요. 블로그는 시험에 출제될 수 있는 보안 관련 주제를 자주 다룹니다
- **최근 CNCF 보안 추가 프로젝트**: 시험 날짜 2-3개월 전에 CNCF 생태계에 합류한 보안 프로젝트에 특별히 주의하세요. 이러한 새로운 보안 도구가 시험에 출제될 수 있으므로 최신 정보를 유지하는 것이 중요합니다
- **Kubernetes 보안 문서**: 공식 Kubernetes 보안 문서, 특히 Pod 보안, 네트워크 정책, RBAC, 시크릿 관리를 검토하세요

**학습 전략:**
- Kubernetes 보안에 이미 익숙하다면 (예: CKS 또는 보안 관련 업무 경험), 준비를 더 효율적으로 완료할 수 있습니다
- "어떻게"보다 보안 "왜"와 "무엇"을 이해하는 데 집중하세요 - KCSA는 4지선다형 객관식 문제로만 개념적 보안 지식을 테스트합니다
- 모의 시험으로 객관식 형식에 익숙해지세요 (기억하세요: 실제 시험에는 실습 과제가 없습니다)
- 클라우드 네이티브 환경의 보안 모범 사례와 일반적인 취약점을 검토하세요
- CNCF 보안 프로젝트와 보안 생태계에서의 역할을 학습하세요

**중요한 알림:**
KCSA 시험은 Kubernetes뿐만 아니라 전체 클라우드 네이티브 랜드스케이프의 보안 측면을 다룹니다. Kubernetes 지식 외에도 컨테이너 보안, 시크릿 관리, 컴플라이언스, 위협 탐지, 보안 도구에 대한 학습 시간을 할당하세요.

## 4. 요약

KCSA 자격증은 클라우드 네이티브 보안을 이해하기 위한 훌륭한 기초를 제공합니다. CKA/CKAD/CKS 보유자와 같이 Kubernetes 실무 경험이 있는 분들에게는 이론적 보안 지식을 검증하고 확장할 기회를 제공합니다. 초보자에게는 클라우드 네이티브 보안 실무에 대한 체계적인 학습 경로를 제공합니다.

KCSA의 주요 특징:
- **입문 수준 보안 자격증**: 클라우드 네이티브 보안을 처음 접하는 분들도 접근 가능
- **광범위한 보안 범위**: Kubernetes뿐만 아니라 전체 클라우드 네이티브 생태계의 보안 측면을 다룸
- **이론적 초점**: 보안 개념과 원칙의 이해를 테스트
- **보완적**: CKS와 같은 실습 자격증이나 실무 보안 경험과 잘 어울림

KCSA 자격증은 클라우드 네이티브 보안 기술에 대한 견고한 기초를 검증하며, 더 고급 보안 자격증으로 나아가는 디딤돌 역할을 하거나 클라우드 네이티브 맥락에서 기존 보안 지식을 체계화하는 방법이 됩니다.
