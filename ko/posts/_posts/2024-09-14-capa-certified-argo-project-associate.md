# CAPA: Certified Argo Project Associate (2024)

* UNORDERED TOC
{:toc .large-only}

## 1. 자격증 소개

Certified Argo Project Associate (CAPA)는 2024년에 새롭게 생긴 자격증으로, Argo 프로젝트 생태계 전반에 대한 전문성을 검증합니다. Kubernetes 클러스터를 강력한 DevOps 플랫폼으로 탈바꿈시키는 4개의 Argo 프로젝트 전체를 다룹니다:

- **Argo Workflows**: ML 파이프라인과 CI/CD 파이프라인을 포함한 범용 워크플로우 오케스트레이션 엔진
- **Argo CD**: 선언형 지속적 배포 도구
- **Argo Rollouts**: 고급 배포 업데이트 전략 관리 도구
- **Argo Events**: 이벤트 기반 Kubernetes 자동화 도구

시험은 60개의 객관식 문제를 90분 안에 풀어야 하며, 합격 커트라인은 75%입니다. 자격증 유효기간은 2년이며, PSI Secure Browser 환경에서 감독관이 있는 방식으로 진행됩니다.

## 2. 시험 후기

![CAPA 자격증](/assets/img/certtifications/7528f6b6-eab2-4716-a7f2-227a331a7e8f.png){:.lead width="500" height="100" loading="lazy"}

CAPA 자격증 2024. 2024년 9월 14일 발급. 2026년 9월 15일 만료. 자격증 확인은 [여기](https://www.credly.com/badges/ee42c2c7-2ac3-411f-8713-cc26cbec8022)에서 할 수 있습니다.
{:.figcaption}

2년간 실무에서 Argo 프로젝트들을 적극 활용한 후 이 자격증을 취득했습니다. 실무에서의 경험은 다음과 같습니다:

- 데이터 파이프라인과 CI/CD 파이프라인에 Argo Workflows 활용
- 웹 애플리케이션 및 Kubernetes 리소스 배포에 Argo CD 도입
- Argo Rollouts로 배포 전략 관리
- Argo Events로 커스텀 CI/CD 이벤트 소싱 구축

자격증을 취득하는 과정에서 다음과 같은 도움을 받았습니다:
- 실무 경험으로 쌓은 지식을 체계적으로 정리
- 이전에 사용해보지 못한 숨겨진 기능들을 발견
- 공식 문서를 더 깊이 이해
- 현재 구현 방식에서 개선할 부분 파악

시험을 위해 다음 항목에 집중해서 준비했습니다:
1. 4개 Argo 프로젝트의 공식 문서 철저히 학습
2. 로컬 Kubernetes 환경에서 다양한 기능 직접 구현 및 테스트
3. 여러 Kubernetes 시뮬레이터(docker-desktop, kind, microk8s, orbstack)로 실습

시험 난이도는 CKAD와 비슷한 수준으로, 실무 경험이 있다면 어렵지 않게 도전할 수 있지만, 도구 전반에 대한 폭넓은 지식은 필요합니다.

## 3. 요약

CAPA 자격증은 Argo 생태계에 대한 실용적인 지식을 효과적으로 검증합니다. 객관식 시험이지만 실제 환경에서 Argo 컴포넌트들이 함께 동작하는 방식에 대한 깊은 이해가 필요합니다.

자격증을 준비하는 분들을 위한 팁:
- 가능하다면 공식 강좌를 적극 활용하세요
- 덜 알려진 기능을 포함해 4개 Argo 프로젝트 전부를 실습하세요
- 시험에 자주 출제되는 GUI 관련 문제를 잘 챙기세요
- 실무에서 사용해보지 않았더라도 엔터프라이즈 기능을 공부해두세요

앞으로 출시될 고급 Argo 자격증이 있다면 꼭 도전해보고 싶습니다. 이 자격증을 취득하면서 쌓은 핸즈온 경험이 이미 팀의 Argo 프로젝트 활용 방식을 개선하는 데 도움이 됐습니다.

CAPA 자격증을 고려하는 분들께 이 글이 도움이 되길 바랍니다. Argo 프로젝트는 Kubernetes 생태계에서 점점 더 중요해지고 있으며, 이 자격증은 관련 도구들에 대한 전문성을 입증하는 의미 있는 방법입니다.
